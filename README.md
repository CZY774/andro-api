# Helloworld Android

A simple Android application built with Kotlin and Java using Gradle and Retrofit.

## Features

- Fetch and display user data from a remote API using Retrofit
- MVVM architecture with LiveData and ViewModel
- Modern Android UI components

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CZY774/andro-api.git
   ```
2. **Open with Android Studio**
3. **Build and run the app** on your Android device or emulator

## Project Structure

- `app/src/main/java/com/czy/helloworldandroid/` — Main source code
- `app/src/main/res/layout/` — Layout XML files
- `app/src/main/AndroidManifest.xml` — App manifest

## Dependencies

- Kotlin
- Java
- AndroidX
- ViewModel & LiveData
- Retrofit
